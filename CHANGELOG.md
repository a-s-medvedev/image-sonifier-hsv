# Changelog

## v0.3.1

- Улучшена загрузка пользовательских изображений на iPhone и iPad: для iOS используется FileReader, изображение удерживается в памяти до завершения декодирования, добавлен таймаут и понятное сообщение для неподдерживаемых форматов.
- Обновлена stand-alone сборка с тем же исправлением.

## v0.3.0

- Перевод интерфейса на русский язык.
- HSV-модель тембра: оттенок управляет характером тембра, насыщенность - гармонической сложностью, value - громкостью.
- Добавлено зацикливание, кеширование подготовленного аудио, индикатор готовности и анимация прохода по спектрограмме.
- Улучшена мобильная компоновка, добавлен лимит обработки больших изображений до 2048 px.

## v0.2.0

- Добавлены Цветной и Художественный режимы.
- Добавлены настройки частотного диапазона, контраста, громкости, инверсии яркости и обратного времени.
- Добавлен экспорт WAV.

## v0.1.0

- Первый прототип: загрузка изображения, Canvas-обработка, преобразование яркости в амплитуду и Web Audio-синтез.
