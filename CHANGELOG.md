# Changelog

## v0.3.4

- Возвращены отдельные кнопки «Играть» и «Стоп» из последней стабильной схемы воспроизведения.
- Подготовка WAV снова начинается только по явному нажатию «Играть» и не зависит от системных контролов браузера.
- Удалена экспериментальная автоматическая подготовка звука из v0.3.3.
- Основная цель выпуска — восстановить одинаковое поведение на Android и настольных браузерах.
- Кнопка «Играть» запускает готовый буфер через Web Audio после разблокировки контекста в момент нажатия, поэтому долгий расчёт не требует второго клика.

## v0.3.3

- Убран перехват нажатий внутри нативного аудиоплеера, из-за которого Play не работал в Safari и на iPhone.
- WAV теперь автоматически готовится после загрузки изображения и после изменения параметров.
- Повторные изменения объединяются debounce-таймером, а устаревшая генерация корректно отменяется.
- Нативный Play запускает только уже подготовленный файл и сохраняет realtime-тембральную цепочку.

## v0.3.2

- Настройки сигнала больше не запускают повторную HSV-обработку и перерисовку изображения.
- Исходные пиксели хранятся в компактном байтовом формате, что заметно снижает расход памяти на больших изображениях.
- Обработанные HSV-данные больше не содержат неиспользуемый четвёртый канал.
- Внутренний цикл синтеза больше не создаёт временные объекты для каждого аудиосэмпла.
- Цветная спектрограмма отрисовывается через готовую таблицу цветов без миллионов временных массивов.
- Исправлено двойное применение громкости в цепочке HTML Audio и Web Audio.
- Аудиограф повторно используется при воспроизведении подготовленного файла; удалён неиспользуемый код старого плеера.
- Научный режим больше не получает скрытое искажение от realtime-цепочки тембра.
- Строка флажков под спектрограммой компактнее помещается на экранах телефонов.
- К CSS и JavaScript добавлена версия, чтобы Safari не использовал устаревшие файлы из кеша после обновления.

## v0.3.1

- Блок воспроизведения переведен на HTML audio-плеер с `playsinline`, как в сонификаторе данных.
- Повторный Stop/Play использует подготовленный WAV, если настройки не менялись.
- Спектрограмма переименована в "Спектрограмма"; исходное изображение снова расположено выше спектрограммы.
- Блоки "Тембр" и "Сигнал" поменяны местами.
- На узких экранах сверху показываются спектрограмма и изображение, ниже - все элементы управления.
- При сужении экрана блок "Изображение" поднимается выше карточки исходного изображения.
- При сужении экрана все блоки растягиваются на ширину контейнера.
- Убран лишний текст статуса при воспроизведении, пояснение тембра перенесено под ползунки.
- В блок "Модель" добавлен выбор ограничения размера изображения: 2k, 1024 или без ограничений.
- На узких экранах кнопки воспроизведения остаются в один ряд.
- Архивы выпускаются с номером версии в имени файла.

## v0.3.0

- Перевод интерфейса на русский язык.
- HSV-модель тембра: оттенок управляет характером тембра, насыщенность - гармонической сложностью, value - громкостью.
- Добавлено зацикливание, кеширование подготовленного аудио, индикатор готовности и анимация прохода по спектрограмме.
- Тестовое изображение создаётся через Canvas, чтобы стартовая версия не зависела от загрузки PNG в Safari.

## v0.2.0

- Добавлены Цветной и Художественный режимы.
- Добавлены настройки частотного диапазона, контраста, громкости, инверсии яркости и обратного времени.
- Добавлен экспорт WAV.

## v0.1.0

- Первый прототип: загрузка изображения, Canvas-обработка, преобразование яркости в амплитуду и Web Audio-синтез.
