# Сонификатор изображений

Браузерный учебный инструмент для science/art воркшопов: изображение читается как спектрограмма и превращается в звук через Web Audio API.

## Идея

- X = время.
- Y = частота.
- Value / яркость = громкость.
- Hue / оттенок = характер тембра.
- Saturation / насыщенность = гармоническая сложность.

Геометрия изображения задает структуру звука, а цвет меняет характер звучания без прямого перевода цвета в ноты.

## Как запустить

Откройте `index.html` в современном браузере. Сервер и сборка не нужны.

Для телефонов лучше использовать Safari на iOS или Chrome на Android. Первое воспроизведение должно запускаться нажатием кнопки, потому что мобильные браузеры блокируют автоматический старт аудио.

## Основные возможности

- Загрузка изображения с локального устройства.
- Встроенное тестовое изображение по умолчанию.
- Цветная спектрограмма с осями времени и нотами C по частоте.
- Научный, Цветной и Художественный режимы.
- Кеширование подготовленного звука: повторное прослушивание без изменений не пересчитывает спектрограмму.
- Ограничение обработки больших изображений до 2048 px по длинной стороне для устойчивой работы в браузере.
- Экспорт результата в WAV.

## Совместимость

Приложение использует Canvas API, File API и Web Audio API. Рекомендуются актуальные версии Chrome, Edge, Firefox и Safari.
